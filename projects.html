E-Commerce Accessibility Overhaul

HTML5/ARIA Vanilla JS WCAG Audit

The Challenge: A large enterprise checkout flow was completely unusable for keyboard-only and screen reader users due to dynamic modal updates failing to move system focus cleanly.

The Solution: Structured dynamic sub-sections using proper landmark regions, utilized JavaScript focus management to move focus intentionally to opened states, and implemented aria-live="polite" for non-disruptive cart total alerts.

Open Source Design System

Web Components CSS Custom Properties A11y Testing

The Challenge: Teams across organization groups were consistently recreating inaccessible modal accordions and select lists, introducing compliance issues.

The Solution: Formulated an open-source structural component repository with baked-in keyboard arrow-key navigation handlers and proper semantic token configurations ensuring automated AA color contrast compliance ratios natively.
